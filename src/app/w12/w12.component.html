Internet of things (IOT)

About IOT :- If you think the Internet has changed your life, think again. IOT gonna change it again! It is a global network infrastructure connecting physical and virtual objects using cloud computing, data capture and network communication. It allows devices to communicate with each other, access information on the Internet, store and retrieve data, and interact with users, creating smart, comprehensive and always-connected environments. A basic example of this, imagine you forgot to switch off the lights/fan/AC, while leaving for the work, don’t worry IOT will enable to switch it off using your mobile.
